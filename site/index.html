Project Highlights

Exhibit: a lightweight data publishing framework that you can use to create rich mash-ups (see screenshot below) using just HTML and JSON. No database, no web application whatsoever.

Picker: a web interface for students to select classes, built on top of Exhibit. It was spotlit on MIT's front page.

Timeline: the equivalence of Google Maps but for temporal data—the first Web API for rendering interactive timelines in web pages.
